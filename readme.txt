== Description ==

Simple Shopping Cart Plugin For WordPress Mu. Optional BuddyPress.

<a href="http://eos.exbridge.jp/versions/download/18?attachment_id=25">全体概要（画面遷移）</a>
<a href="http://eos.exbridge.jp/versions/download/18?attachment_id=26">WP-Simple-Cart導入ガイド</a>

WordPressを使ってECサイトを構築するためのプラグインです。
Mu版のWordPressを前提条件とし、ショッピングカート機能を実現しています。
オプション設定でBuddyPressと連携することも可能です。
